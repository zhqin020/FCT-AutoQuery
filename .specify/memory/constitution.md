### I. Environment Activation
Before starting console or running commands, the environment must be activated: conda activate fct

### II. Issue Tracking
Upon completion of problem resolution, record and summarize in the issues directory, update the issue file, and continue resolution until the status is CLOSED

### III. Logging
Project logs are located in the logs directory, with the latest log file being: logs/scraper-1.log

### IV. Testing
Test files must only be stored in the tests directory, named as test_*.py
